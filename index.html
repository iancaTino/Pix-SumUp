O que são as chaves Pix

Você poderá fazer um Pix usando seus dados bancários. Mas com as chaves Pix isso fica ainda mais simples. Para receber transferências ou pagamentos, você precisará passar uma única informação, a chave Pix, assim o dinheiro chega na sua conta em segundos.

Suas chaves Pix podem ser:

  • CPF ou CNPJ

  • E-mail

  • Número de celular

  • Chave aleatória

  • Para criar o QR Code para receber suas vendas, você precisa associá-lo à uma de suas chaves Pix.

Perguntas frequentes

Como funciona o Pix? ▾

Criado pelo Banco Central, o Pix chega para tornar o processo de transferir dinheiro muito mais rápido. Diferente de outras formas de transferência como TED, DOC ou até débito, com o Pix não importa se é final de semana, feriado ou até tarde da noite: o dinheiro cai na conta em poucos segundos.

O que é uma Chave Pix? ▾

A chave Pix é uma identificação única que você pode cadastrar para receber transferências pelo Pix. Ela pode ser um número de celular, CPF, CNPJ, e-mail ou chave aleatória. Isso facilita o processo, pois você não precisa mais informar dados bancários completos para realizar uma transferência.

O que é uma chave aleatória? ▾

A chave aleatória é uma chave gerada automaticamente pelo sistema do banco, de forma única, para você. Ela não depende de dados como CPF ou e-mail, e oferece maior segurança nas transações, já que não é possível identificar o proprietário diretamente.

Posso cadastrar a mesma Chave Pix em vários bancos? ▾

Não, cada chave Pix é vinculada a uma conta bancária específica. No entanto, você pode cadastrar diferentes chaves para diferentes contas ou até mesmo usar uma chave aleatória que não esteja vinculada a um banco específico.
